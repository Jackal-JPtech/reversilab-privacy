# プライバシーポリシー (Privacy Policy)

「ReversiLab」（以下、「当アプリ」といいます）は、ユーザーの皆様のプライバシーを尊重し、個人情報の保護に努めています。本プライバシーポリシーでは、当アプリにおける情報の収集および利用方針について説明します。

## 1. 情報の収集について

当アプリは、ユーザーの氏名、住所、電話番号、メールアドレスなどの個人を特定できる情報（生年月日や連絡先など）を独自に収集または送信することはありません。

## 2. デバイス内のデータについて

当アプリ内のプレイ進行状況（道場のクリア状況、対戦記録、ポイント、設定など）は、ユーザーのデバイス（スマートフォンやタブレット等）のローカルストレージ内にのみ保存されます。これらのデータが自動的に外部のサーバーへ送信されることはありません。

## 3. 分析ツールと広告配信について

当アプリでは、サービス向上のための使用状況分析や、無料でのサービス提供を維持するための広告の表示を目的として、サードパーティが提供するツールを利用する場合があります。これにより、個人を特定しない範囲のデバイス情報や利用状況が収集されることがありますが、それらが個人情報と紐付けられることはありません。

## 4. プライバシーポリシーの改定

本プライバシーポリシーは、法令の変更や機能追加などに伴い、事前通知なく改定される場合があります。最新の方針については、アプリ内または本ページにて随時公開いたします。
